# FL Studio ASIO Driver Manager
A simple tool to install, update, or remove FL Studio’s ASIO drivers.

## Usage

1. Run `FL Studio ASIO Driver Manager.bat`
2. Select an option from the menu:
   - Install or update driver
   - Uninstall driver
   - Exit
